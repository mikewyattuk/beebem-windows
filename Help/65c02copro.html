Acorn 65C02 Co-Processor

BeebEm emulates the Acorn 65C02 Co-Processor board. This co-pro contains a 3MHz 65C02 processor with 64KB of RAM. It uses the BBC for all input/output via the 'tube' interface. The co-pro runs Hi-Basic which makes more RAM available for programs than is available normally.

The 65C02 co-pro also runs the tube version of Elite! Go to the Software section of the BeebEm website for a link.
